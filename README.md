# Calculadora de Retrabalho

Esta é uma aplicação web desenvolvida em **Python** utilizando a biblioteca **Streamlit**. O objetivo da ferramenta é automatizar o cálculo financeiro do retrabalho em diferentes processos industriais, permitindo uma gestão de custos mais ágil e precisa.

---

## Funcionalidades

* **Seleção de Setores:** Interface intuitiva para escolher o setor específico (Solda, Pintura, Usinagem, etc.).
* **Cálculo Automático:** Converte o tempo gasto (em minutos) no valor monetário correspondente com base no custo-hora do setor.
* **Visualização de Métricas:** Exibição clara do custo final e do custo base utilizado na conta.
* **Layout Limpo:** Organização em colunas para melhor usabilidade em desktops e dispositivos móveis.

---

## Tecnologias Utilizadas

* **Python:** Linguagem base do projeto.
* **Streamlit:** Framework para criação de dashboards e aplicações web.

---

## Como Executar a Aplicação

### 1. Pré-requisitos
Certifique-se de ter o Python instalado. Você precisará instalar a biblioteca Streamlit via terminal:

```bash
pip install streamlit
```
### 2. Execução
Com o arquivo do código (ex: app.py) na sua pasta, execute no terminal:
```bash
streamlit run app.py
```
